# Kuwo Music Downloader PRO
## 使用方法
1. 打开酷我音乐网页版；
2. 点击搜索按钮，搜索想要的歌曲；
3. 进入试听页面；
4. 复制试听页链接（浏览器上方），启动Kuwo Musices Downloader Pro
5. 将复制的链接粘贴到程序【音乐链接】输入框；
6. 点击提交； 
7. 点击下载；
8. 完成！
